## 🚀 如何初始化假資料

### 方法 1: 透過 API 端點

在瀏覽器中訪問：
```
http://localhost:3000/api/init-fake-data
```

或使用 curl：
```bash
curl http://localhost:3000/api/init-fake-data
```

### 方法 2: 自動初始化

假資料會在以下情況自動初始化：
- 當用戶數量少於 2 個時（只有管理員）
- 當產品數量少於 15 個時
- 當訂單數量為 0 時

## 📝 注意事項

- 假資料只會在資料庫為空或資料不足時建立
- 不會重複建立已存在的用戶或產品
- 所有密碼都已加密儲存
- 訂單會使用現有產品建立

## 🔄 重置假資料

如需重置假資料，可以：
1. 刪除 `data/` 目錄下的 JSON 檔案
2. 重新訪問初始化 API 端點
